######################### Работа с MongoDB
sudo service mongod start - запуск базы
sudo service mongod status - проверка статуса базы

use test_db - первая команда
db.createCollection("users") - создание коллекции
db.collection_name.drop() - удаление коллекции

##################################

nodemon - пакет для сохранения изменений на лету
body-parser - парсит данные в JSON формат

morgan - позволяет логировать запросы
cors - пакет для удобного ответа, если клиент делает запрос с другого сервера
Эти два пакета чисто для удобства разработки

mongoose - пакет для работы с БД
bcryptjs - пакет для шифрования паролей
jsonwebtoken - пакет для генерации токена
passport.js - защита роутов. Также регистрация в соц сетях. Реализация стратегий

moment, multer - библиотеки для работы с файлами
